常用命令  
-------    
1.本地同步仓库内容：  
git pull git@github.com:luckyqsz/ASAGroup.git  
2.增加文件：  
git add .  
（注意add和.之间有空格，加.是提交所有文件，提交指定文件可以写文件名）  
3.提交文件  
git commit -m "这里写下你自己的记录本次提交内容的信息"  
4.推送到远程仓库  
git push git@github.com:luckyqsz/BCRC-ASAGroup.git
